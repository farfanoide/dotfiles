----[ Installation: ]----------------------------------

	# clone the repo
	> git clone https://github.com/farfanoide/dotfiles

	# cd into it
	> cd dotfiles

	# this will make symbolic links to your home folder, deleting any previous config files in there
	> ./link.sh
